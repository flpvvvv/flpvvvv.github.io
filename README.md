# flpvvvv.github.io

Personal blog and portfolio site for flpvvvv - a data scientist exploring statistics, machine learning, and data science.

Built with [Astro](https://astro.build) and deployed on [GitHub Pages](https://pages.github.com/).

## Site

- **URL**: https://flpvvvv.com
- **Author**: flpvvvv

## Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

## Tech Stack

- **Framework**: [Astro](https://astro.build) v5
- **Styling**: [Tailwind CSS](https://tailwindcss.com)
- **Math Rendering**: [KaTeX](https://katex.org) via rehype-katex
- **Syntax Highlighting**: Shiki (GitHub Dark theme)
- **Deployment**: GitHub Actions → GitHub Pages

## Content

Blog posts covering statistics and data science topics, primarily notes from the **MITx: 18.6501x Fundamentals of Statistics** course on edX.

Topics include:
- Probability theory and distributions
- Estimation methods (MLE, Method of Moments, M-estimation)
- Hypothesis testing
- Bayesian statistics
- Confidence intervals
